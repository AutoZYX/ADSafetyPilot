# Chinese L2 ADAS Safety Requirements — Test Scenarios

Source: 《智能网联汽车 组合驾驶辅助系统安全要求》(公开征求意见稿, 2024-09)

## Standard Overview

This is a **mandatory national standard (GB)** covering L2 combined driving assistance systems. It includes both technical requirements and test methods.

## Test Scenario Categories

### 7.1 基本功能试验 (Basic Function Tests)
- ACC activation/deactivation
- System state transitions

### 7.2 纵向控制试验 (Longitudinal Control Tests)
- ACC steady-state following
- Lead vehicle deceleration response
- Lead vehicle stop & go

### 7.3 横向控制试验 (Lateral Control Tests)
- Lane keeping on straight road
- Lane keeping on curves
- Lane departure warning

### 7.4 紧急制动试验 (Emergency Braking Tests)
- AEB for stationary vehicle
- AEB for moving vehicle
- AEB for pedestrian crossing
- AEB for cyclist

### 7.5 综合场景试验 (Complex Scenario Tests)
- 7.5.1 Cut-in scenarios (前车切入)
- 7.5.2 Roundabout navigation (环岛通行)
- 7.5.3 Traffic light recognition (信号灯识别与响应)
- 7.5.4 Vehicle target detection (车辆目标探测与响应)
  - 7.5.4.1 Curve stationary vehicle (弯道静止车辆)
  - 7.5.4.2 Intersection conflict - straight (交叉口冲突-直行)
  - 7.5.4.3 Intersection conflict - crossing (交叉口冲突-横穿)
  - 7.5.4.4 Stationary vehicle with oncoming traffic (静止车辆+对向车)
- 7.5.5 Motorcycle target detection (摩托车目标探测)
- 7.5.6 Pedestrian/Cyclist tests (行人/自行车试验)

### 7.6 驾驶员监控试验 (Driver Monitoring Tests)

### 7.7 系统安全试验 (System Safety Tests)
- Functional safety requirements
- SOTIF requirements
